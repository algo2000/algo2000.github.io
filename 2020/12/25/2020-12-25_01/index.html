변수 선언문

let과 const

기존 자바 스크립트 es5에서는 var 이라는 것을 이용하여 변수를 선언했습니다.

하지만 현재 자바스트립트에서는 let과 const를 이용하여 변수를 선언합니다.

  • let

    위와 같은 방식으로 사용을 할 수 있습니다. let은 수시로 바뀔 수 있는 값입니다.

  • const

    위와 같은 방식으로 사용을 할 수 있습니다. const는 초기값이 정해지면 절대 변하지 않습니다.

    const 이용시 반드시 초기값을 세팅해 줘야합니다.

어떤 값이든 저장할 수 있도록 타입을 명시할 수 있습니다.

undefined

js에서는 값을 초기화 하지 않은 경우 undefined라는 값을 가집니다.

typescript의 undefined 타입은 undefined 값만을 가집니다.

그렇기 때문에 그 이외의 값이 들어오면 오류가 발생합니다.

템플릿 문자열

변수의 값을 문자열로 만들 수 있게 해주는 템플릿 문자열을 제공합니다.

`(역 따옴표)로 작성하고자하는 문자열을 감싸서 작성합니다.

${[변수이름]} 이렇게 구성하여 변수의 있는 값으르 사용할 수 있습니다.

객체, 인터페이스

object의 하위 타입에는 interface, class가 있습니다.

object 타입은 number, boolean, string 타입을 가질수 없지만 나머지 다른 객체들을 모두 가질 수 있습니다.

이처럼 object는 객체에서 any의 역할을 수행합니다.

interface

interface에서는 해당 객체의 변수를 타입 주석의 형식을 나열하여 사용합니다.

interface의 조건

인터페이스는 여러가지 조건이 존재합니다.

  1. 없는 속성을 세팅한 경우
  2. 나열한 속성을 제외하고 세팅한 경우
