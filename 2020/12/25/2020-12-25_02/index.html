함수 표현식

함수 = 객체 (function = object)

위의 예시와 아래 예시는 같은 동작을 합니다.

function(a, b) {return a + b}처럼 표현하는 것을 함수 표현식이라고 합니다.

함수 표현식은 함수형 언어(자바스크립트)의 핵심기능 입니다.

일등 함수

언어가 일등 함수를 제공한다면 함수형 프로그래밍 언어 입니다.

처음에는 a + b라는 표현식을 저장하였지만 이후에 a - b라는 표현식을 대입합니다.
이처럼 변수인지 함수인지 차별하지 않고 담습니다.

계산법

컴파일러가 1 + 2 라는 표현식을 만나면 조급한 계산식을 적용해 3이라는 값을 만듭니다.
컴파일러가 function(a, b) {return a + b}를 만나면 a와 b의 값을 알 수 없어 느긋한 계산법을 적용하여 계산을 보류합니다.

실행문과 표현식 문

실행문은 CPU에서 실행되는 코드를 의미합니다.
실행문은 CPU에서 실행만 될 뿐 결과를 알려주지 않습니다.
실행문이 실행된 결과를 알려면 반드시 return 키워드를 사용해야 합니다.
반면 표현식 문은 CPU에서 실행된 결과를 return 키워드를 사용하지 않아도 알려줍니다.
